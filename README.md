# Como eliminar cuenta en QueHago!

## Pasos para eliminar tu cuenta:

1. Abre la aplicación en tu dispositivo móvil.
2. Inicia sesión con tus credenciales.
3. Ve a la sección de "Configuración".
4. Selecciona la opción "Eliminar cuenta".
5. Ingresa tus credenciales nuevamente "Correo" y "Contraseña".
6. Confirma que deseas eliminar tu cuenta.

7. **Nota:** La eliminación de la cuenta es permanente y no se puede revertir. Todos tus datos serán eliminados de manera irreversible.
